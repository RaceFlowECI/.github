## Proyecto

**RaceFlow** es una plataforma web de salas de entrenamiento colaborativas en tiempo real
para deportes de cronometraje, desarrollada como proyecto de *Arquitecturas de Software (ARSW)*
en la Escuela Colombiana de Ingeniería Julio Garavito.

Ver todos los repositorios: [github.com/RaceFlowECI](https://github.com/RaceFlowECI)
